神经网络（NN：Neural Networks）

多层前向神经网络：
输入层（特征向量值）
隐藏层（可以有多层）
输出层（标记分类）
权重
偏置
激励函数activation function
隐藏层足够多，训练集足够大，可以模拟任何情况

NN算法（BP）
1. 特征向量标准化normalize（转化为0-1之间的值）
2. 随机初始化权重和偏置（-1----1）
3. 传入样例
4. 计算神经元的值
    a. 对上层输入加权求和
    b. 加上偏置值
    c. 用激励函数产生最后结果（sigmoid function）：f(x)=1/(1+e^(-x))
5. 根据误差反向传输
http://blog.csdn.net/acdreamers/article/details/44657439
6. 误差计算
    a. 输出层Err=(结果)(1-结果)(预期值-结果)
    b. 隐藏层Err=(结果)(1-结果)(后一层Err加权求和)
    c. 权重更新
    d. 偏置更新


训练NN
1. 不断通过样例调整权重和偏向的过程
2. 训练好的神经网络可以用于预测新的样例

训练终止
1. 一定次数
2. 既定错误率
3. 权重更新低于某个Threshold

激励函数
1. 双曲函数
2. 逻辑函数
3. 阶跃函数
